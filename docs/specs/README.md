# 仕様ドキュメント

## 概要

本ディレクトリでは、変更頻度の高いモジュールの要件定義と設計仕様を提供します。
要件（requirements.md）と設計（design.md）をセットで管理し、実装の意図を明示しています。

基盤モジュール（error, fs, log, model, protocol, cli）の設計ドキュメントは [`docs/internal/`](../internal/README.md) で管理します。
