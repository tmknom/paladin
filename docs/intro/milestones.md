# Paladin マイルストーン定義

## 1. この文書の役割

この文書は、Paladin の価値成立の節目を管理するものです。
実装セッションの管理には使わず、「何が成立したら次の段階に進めるか」という大粒度の境界だけを定義します。

日々の実装順序は `roadmap.md` で管理します。

## 2. マイルストーン一覧

| ID | タイトル | 含むロードマップ項目 |
| --- | --- | --- |
| M1 | 契約成立 | R-000 |
| M2 | 解析入口成立 | R-010, R-020 |
| M3 | 診断器 MVP 成立 | R-030, R-040 |
| M4 | ルール拡張基盤成立 | R-050〜R-052, R-060〜R-064 |
| M5 | AI 連携強化 | R-070 |
| M6 | 実運用対応 | R-080〜R-083, R-090〜R-093 |
| M7 | 高度解析と修正支援 | R-100〜R-103, R-110〜R-112 |

## 3. 各マイルストーン

### M1. 契約成立

#### 到達目標

Paladin が返すべき結果の形が固定されている状態。

#### 含むロードマップ項目

- R-000 契約モデルを定義する

#### 完了条件

- `status` の 3 状態が定義済みである
- 終了コード仕様が定義済みである
- `Violation / CheckSummary / CheckResult` が定義済みである
- 出力契約の骨格が文書化済みである

---

### M2. 解析入口成立

#### 到達目標

対象ファイルを列挙し、AST を作れる状態。

#### 含むロードマップ項目

- R-010 Python ファイルを再帰列挙する
- R-020 AST を生成する

#### 完了条件

- `.py` ファイルを再帰列挙できる
- 各ファイルの AST を生成できる
- 失敗を `status=error` として扱える

---

### M3. 診断器 MVP 成立

#### 到達目標

1 本のルールで違反検出し、固定契約で返せる状態。

#### 含むロードマップ項目

- R-030 ルール実行基盤を作る
- R-040 `check` コマンドの text 出力と終了コードを実装する

#### 完了条件

- ルール 1 本が動作する
- `Violation` を複数件返せる
- `check` が text 出力を返せる
- 終了コード 0 / 1 / 2 が機能する

---

### M4. ルール拡張基盤成立

#### 到達目標

ルールを継続的に増やしていける状態。

#### 含むロードマップ項目

- R-050〜R-052 単一ファイルルールの追加
- R-060 `rules` コマンドの最小実装
- R-061 `version` コマンドの実装
- R-062 `rules --rule <RULE_ID>` の対応
- R-063 `explain` 導入の再判断
- R-064 `explain` の text 出力実装 (R-063 で導入を選択した場合)

#### 完了条件

- 複数ルールを安定追加できる
- `rules` でルール一覧を見られる
- `version` で実行バージョンを確認できる

---

### M5. AI 連携強化

#### 到達目標

生成 AI や他ツールから JSON 診断を安定利用できる状態。

#### 含むロードマップ項目

- R-070 `--format json` を正式対応する

#### 完了条件

- `--format json` が正式利用可能である
- JSON 構造が安定している
- 出力を後続処理へ安全に渡せる

---

### M6. 実運用対応

#### 到達目標

例外許容やプロジェクト差異を吸収しながら使える状態。

#### 含むロードマップ項目

- R-080〜R-083 Ignore 機能
- R-090〜R-093 設定ファイル

#### 完了条件

- Ignore が使える
- 設定ファイルでルール制御できる
- 対象除外やルール調整が可能である

---

### M7. 高度解析と修正支援

#### 到達目標

単一ファイルを超える設計ルールと、修正支援へ進める状態。

#### 含むロードマップ項目

- R-100〜R-103 複数ファイル解析
- R-110〜R-112 修正支援

#### 完了条件

- 複数ファイル解析ルールを実装できる
- 修正方向性が高度化されている
- 必要に応じて限定的自動修正を検討できる
